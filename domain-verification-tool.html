🚨 PROBLEM: İnternet Siteleri Bulunamadı

⚠️ DURUM: Domain verification kodu eksik veya yanlış. Bu yüzden "İnternet siteleri" bölümü bazen görünüyor bazen gitmiyor.

📋 Hızlı Çözüm Adımları:

1. Meta Business Manager'dan verification kodunu alın

2. Aşağıya yapıştırın

3. Güncellenmiş kodu kopyalayın

4. GitHub'da index.html'e yapıştırın

🔗 1. ADIM: Meta Business Manager

Aşağıdaki linke tıklayın:

🌐 Domain Settings'i Aç

Domain ekleyin:

• "Add Domain" → kargonburada.online

• "Add Meta Tag" seçeneğini seçin

• Verilen kodu kopyalayın

📝 2. ADIM: Verification Kodunu Yapıştırın

✅ 3. ADIM: Güncellenmiş index.html Kodu

✅ KOD HAZIR! Aşağıdaki kodu kopyalayın ve GitHub'da index.html dosyanızın başına yapıştırın.

📋 Yapılacaklar:

1. ✅ Yukarıdaki kodu kopyaladınız

2. 🌐 GitHub'da index.html dosyasını açın

3. 🔄 Mevcut <head> bölümünü değiştirin

4. 💾 Commit edin ve deploy bekleyin (2-3 dakika)

5. 🔍 Meta'da "Verify Domain" butonuna basın

⏱️ 4. ADIM: Bekleme ve Kontrol

🕐 Süreç:

• Deploy: 2-3 dakika (Vercel otomatik)

• Meta Tarama: 5-10 dakika

• Doğrulama: Anında

✅ Başarı Kontrolü:

• Meta Business Manager'da ✅ işareti görünecek

• Events Manager → "İnternet siteleri" bölümünde site görünecek

• Artık sabit kalacak, gidip gelmeyecek
