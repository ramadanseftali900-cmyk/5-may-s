⚙️ 3. ADIM: Manuel Event Ayarları

🛒 Purchase Event Ayarları

Tetikleyici: Sipariş başarı sayfası veya "Sipariş Ver" butonu

Parametreler:

value: Sipariş tutarı (örn: 299.90) currency: TRY content_ids: Ürün ID'si content_name: Ürün adı content_type: product num_items: Adet

👁️ ViewContent Event Ayarları

Tetikleyici: Ürün sayfası yüklendiğinde

Parametreler:

value: Ürün fiyatı currency: TRY content_ids: Ürün ID'si content_name: Ürün adı content_type: product

🛍️ AddToCart Event Ayarları

Tetikleyici: "Sepete Ekle" veya "Sipariş Ver" butonu

Parametreler:

value: Ürün fiyatı currency: TRY content_ids: Ürün ID'si content_name: Ürün adı content_type: product quantity: Adet

🧪 4. ADIM: Test ve Doğrulama

🔍 Event Test Etme

1. Event Setup Tool'da "Test" butonuna basın

2. Sitenizde işlem yapın (ürün görüntüle, sipariş ver)

3. Events Manager'da "Test Events" sekmesinde kontrol edin

✅ BAŞARILI KURULUM İÇİN:
• Purchase eventlerinde value ve currency parametreleri doğru olmalı
• ViewContent eventleri ürün sayfalarında tetiklenmeli
• AddToCart eventleri sepete ekleme işlemlerinde çalışmalı

🚀 5. ADIM: Gelişmiş Ayarlar

📊 Conversions API Kurulumu

Server-side tracking için Conversions API'yi de kurun:

1. Meta Business Settings → Data Sources → Conversions API 2. Access Token oluşturun 3. Server endpoint'inizi ayarlayın 4. Test edin ve doğrulayın

🎯 Optimizasyon Ayarları

Reklam kampanyalarınızda:

• Conversion Objective: Purchase • Optimization Event: Purchase • Attribution Setting: 7-day click, 1-day view • Value-based bidding: Aktif
